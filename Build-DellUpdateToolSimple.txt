
Simple build steps for DellUpdateToolSimple.exe

1. Save the script as:
   DellUpdateToolSimple.ps1

2. Open PowerShell as Administrator

3. Install PS2EXE:
   Install-Module -Name ps2exe -Scope CurrentUser

4. Build the EXE:
   Invoke-PS2EXE .\DellUpdateToolSimple.ps1 .\DellUpdateToolSimple.exe -title "Dell Update Tool" -noConsole

5. Run:
   DellUpdateToolSimple.exe

What it does:
- Checks whether Dell Command Update is installed
- Opens a small window with buttons
- Check for Updates
- Install Updates
- Open Dell Update

Note:
- The EXE still relies on Dell Command Update already being installed on the PC
- It will ask for Administrator permission when launched
